# Portfolio

Ce dépôt contient une version statique du portfolio basée sur les captures d'écran dans le dossier `Photo`.

Pour voir le site localement, ouvrez `index.html` dans votre navigateur ou servez le dossier avec un serveur local tel que:

```bash
python3 -m http.server 8000
# puis ouvrez http://localhost:8000
```

Pour déployer sur GitHub Pages, activez Pages dans les paramètres du dépôt et choisissez la branche `main` (racine).
